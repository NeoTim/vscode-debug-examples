# VS Code 调试 Jest 测试

## 快速体验

``` bash
git clone git@github.com:forsigner/vscode-debug-examples.git
cd JavaScript/jest-test
npm i
code .  # 在 VScode 打开项目
```

然后，在 VSCode 给代码打上断点，按下 `f5`，看到 Debug toolbar 就说明成功了。
