# Debugging Egg.js App

## Quick experience

``` bash
git clone git@github.com:forsigner/vscode-debug-examples.git
cd JavaScript/egg-app
npm i
code . # open in VScode
```

Then, put a breakpoint on the code in VSCode, press `f5`, and you will see the Debug toolbar.
